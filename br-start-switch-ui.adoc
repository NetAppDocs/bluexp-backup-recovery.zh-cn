== 切换到不同的工作负载

您可以在BlueXP backup and recoveryUI 中切换到不同的工作负载。

.步骤
. 从BlueXP左侧导航中，选择 *保护* > *备份和恢复*。
. 从页面右上角，选择*切换工作负载*下拉列表。
. 选择您想要切换到的工作负载。 image:screen-br-menu-switch-ui.png["BlueXP backup and recovery的交换机工作负载屏幕截图"]
+
页面刷新并在适当的 UI 中显示所选的工作负载。
